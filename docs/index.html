Storage Architecture

MongoDB Atlas

  • Users, Sessions, Tracks
  • Comments, ChatMessages
  • SetlistItems, Feedback
  • JWT auth (bcrypt hashed passwords)

GridFS (MongoDB)

  • Audio blob storage
  • Linked via Track.audioFileId
  • Served via /api/audio/:fileId
  • Supports .webm / .mp4

localStorage (Browser)

  • JWT token (key: 'jwt')
  • Band data (key: 'bands')
  • Fallback tracks & comments when API not configured

Supabase (planned)

  • 001_initial_schema.sql defines tables
  • Mirrors MongoDB schema
  • RLS policies per-user
  • Realtime on comments & chat
